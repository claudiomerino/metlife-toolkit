On design systems

A design system is a unified set of UX and design rules that promote harmony and consitsency across multiple devices and outputs. Elements expected to appear in a design system are:

  • Grid & layout
  • Accessibility guidelines
  • Colors
  • Typography
  • UI components
  • Interaction guidelines

This guide contains real working examples, code snippets, documentation, and style guidelines.
